Image Palette Importer

Instructions

This tool can be used for roughly 3 use cases:

  1. Converting palette images to a Fraymakers palette file
  2. Adding these palette images to an existing Fraymakers palette file
  3. Combining Fraymakers palette files

Can perform all the operations simultaneously

To convert palette images, you need a base/colour mapped image, and then a series of identical images with a different palettes applied
If generating a new palette file, it will need an ID and reference image before being used.
By default, the first image is set as the base palette, you may want to set a different base palette

Tip: You can drag files onto the buttons instead of clicking them.
